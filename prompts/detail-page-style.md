# 详情页统一风格设定模板

## 适用场景
- 护肤品详情页
- 美妆详情页
- 高级感产品详情页
- 需要统一视觉语言的 10 屏分页详情页

## 默认风格
整套分页详情页图片需统一采用高级、克制、干净、理性、温柔的护肤品视觉风格。

整体画面以白色高级留白构图为基础，背景以奶油白、暖米白、浅灰米色等柔和低饱和色调为主，辅以少量咖啡棕与浅金色点缀，营造轻奢杂志感、法式高级感和极简护肤品牌视觉。

整体风格偏：
- clean beauty aesthetic
- editorial skincare campaign
- premium skincare aesthetic
- 高级护肤品牌详情页
- 杂志化页面气质

## 画面表现
- 柔光摄影
- 自然阴影
- 真实材质微距表现
- 包装质感
- 瓶身细节
- 膏体质地
- 肌肤触感
- 高级肤感氛围

## 排版系统
- 大面积留白
- 层级清晰
- 信息克制
- 中文主标题
- 英文辅助标题
- 小字说明
- 图标文案
- 分层信息

## 可使用视觉元素
根据不同页面主题灵活加入：
- 抽象护肤视觉
- 肤感纹理
- 微观结构
- 精华液流动
- 膏体纹理
- 场景道具
- 局部特写
- 包装局部
- 信息图标

## 避免事项
- 杂乱背景
- 强促销感
- 过度饱和
- 低端电商风
- 廉价海报感
- 信息过密
- 满屏重复摆拍
- 每页都只放同一个产品

## 类型适配规则

如果产品不是护肤品，需要根据品类调整：
- 食品：干净高级食品摄影 + 食欲感 + 成分真实感
- 电子：理性科技感 + 材质微距 + 参数清晰
- 家居：生活方式场景 + 材质温度 + 空间质感
- 服饰：面料质感 + 穿搭场景 + 版型细节
- 宠物用品：温暖生活感 + 安全感 + 使用场景
